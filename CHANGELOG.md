# Changelog

All notable changes to MeetMemo will be documented in this file.

## [1.1.0] - 2024-01-15

### Added
- **Compare Page Link**: Added "Compare" link to navigation for easy access to comparison features
- **Blog Section**: New blog section with search functionality
  - Searchable blog posts by title and content
  - Category filtering
  - Responsive grid layout
- **Enhanced Footer**: Updated footer with improved navigation
  - Added Home, Pricing, Compare, and Blog links
  - Added social media links (Twitter, LinkedIn, GitHub)
  - Better responsive design with flex-wrap

### Changed
- Navigation now includes Compare link between Pricing and About Us
- Footer consistency improved with more comprehensive link structure

## [1.0.0] - 2024-01-01

### Added
- Initial release of MeetMemo landing page
- AI-powered meeting assistant features
- Pricing page
- Schedule demo functionality
- Terms of Service and Privacy Policy pages
